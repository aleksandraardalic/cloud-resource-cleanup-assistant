# Cloud Resource Cleanup Assistant

Cloud Resource Cleanup Assistant is a backend application designed to help identify unused or underutilized AWS resources and estimate potential cost savings.

## Features
- Scan stopped EC2 instances
- Generate cleanup reports
- Estimate monthly cost savings
- REST API support

## Tech Stack
- Java 21
- Spring Boot
- AWS SDK v2
- Maven

## Planned Features
- Azure support
- Scheduled scans
- Docker support
- PostgreSQL integration
- Web dashboard
